# Hi! I am Mario 🙋‍♂️

I am a Computer Engineering student at UDC, currently developing my undergraduate final project (TFG). My research focuses on the integration of low-level motion control and computer vision on the Unitree Go2 quadruped platform, utilizing state-of-the-art AI models for autonomous navigation and task execution.

<img src="https://github.com/user-attachments/assets/29f43e69-395a-4c5f-9858-e115d8964ee9" align="right" width="300" alt="hiiiii" style="margin-left: 20px;">

I am a member of a robotics team currently preparing for a local competition at the end of the year. We are currently developing a custom library designed to streamline the iterative coding process for the Olympiad.
This framework simplifies flashing and allows us to change values on-the-fly via Wi-Fi. It operates through a REST API deployed on the robot, which we interface with using an Android app currently under development.

While my current focus is on low-level systems, I aim to become a versatile engineer. To achieve this, I dedicatedly explore and master new technologies every day. I have also been an avid Linux user for the past three years.

Not to brag or anything but, I actually recently won a [hackaton](https://devpost.com/software/k2-platform-hackudc-2026) btw :).
